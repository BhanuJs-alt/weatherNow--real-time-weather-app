# WeatherNow 🌤️

A sleek, real-time weather forecasting application built with a responsive dashboard UI. This project displays current weather statistics, an hourly breakdown, and a 7-day forecast for any searched location using live API data.

## 🚀 Features

- **Real-Time Data**: Fetches live weather status, temperatures, and atmospheric conditions via API.
- **Comprehensive Forecast**: Includes an 8-hour detailed breakdown and a 7-day upcoming forecast.
- **Detailed Metrics**: Displays critical weather parameters including Feels Like temperature, Humidity, Wind speed, and Precipitation.
- **Fully Responsive**: Features a fluid layout that adapts seamlessly from ultra-wide monitors down to mobile screens using CSS Flexbox.
- **Modern Dark Theme**: Styled with a premium, high-contrast dark color palette designed for optimal readability.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Flexbox & Media Queries)
- **Frameworks/Libraries**: Bootstrap 5 (Core layout structure)
- **Scripting**: Asynchronous JavaScript (DOM Manipulation & API Fetching)
